# Phlox Electronic Store

Welcome to Phlox Electronic Store, a fictional electronic store website designed to showcase and sell various electronic products. This project includes HTML and CSS files for the front-end design of the website.

## Features

- **Navigation**: Easy navigation with links to Home, About, Contact, Shop, and Blog sections.
- **Category List**: Browse through different product categories like Computer and Accessories, Camera Audio Video, Smartphone and Phones, etc.
- **Featured Products**: Discover featured products with images, titles, and prices.
- **Most Popular Products**: Explore the most popular products with ratings and prices.
- **Future Offers**: Stay updated with upcoming offers and discounts.
- **Responsive Design**: The website is designed to be responsive and compatible with different devices.

## Technology Used

- **HTML**: Used for structuring the web pages.
- **CSS**: Used for styling the web pages and making them visually appealing.
- **Bootstrap**: Utilized Bootstrap framework (version 5.0.2) for responsive layout and styling components.
- **Font Awesome**: Integrated Font Awesome (version 6.1.2) for adding scalable vector icons to the website.
- **AOS**: Employed AOS (Animate On Scroll) library (version 2.3.1) for smooth animations while scrolling.

## Installation

Simply clone this repository to your local machine to get started:

```git clone https://github.com/MuhammadUmarUsman/phlox-electronic-store ```

## Usage

Open the `index.html` file in your web browser to view the website.
